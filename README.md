## サービス概要
自分のレコードコレクションを「自分だけのこだわり」で管理できる、個人コレクター向けのデジタル書庫アプリです。 既存の海外サービスでは入力が煩雑になりがちな「詳細な盤質ステータス」や「クリーニング履歴」などを、スマホから直感的に記録することに特化しています。 Discogs APIとの連携により、最小限の入力で正確な盤データとジャケット画像を取得し、自分だけの美しいライブラリを構築できます。

## このサービスへの思い・作りたい理由
私自身レコードを収集していますが、世界最大級の管理サービスである「Discogs」は多機能すぎるがゆえに操作が複雑で、英語ベースであることも情報の入力に心理的ハードルを感じさせていました。 また、日本のコレクターにとって重要な「帯の有無」や、細かい「盤質・ジャケの状態」、そして「いつクリーニングしたか」といった二次流通や保存に関わる項目を、手軽かつ詳細に管理したいという欲求がありました。 「自分の持っている盤を正確に把握し、最適な状態で鑑賞・保存したい」という、レコード好きなら誰しもが抱く悩みを技術で解決したいと思い、このサービスを企画しました。

## ユーザー層について
ターゲット：日本の個人レコードコレクター（特に30枚〜数百枚程度の所有者）

理由1： 数千枚を所有するプロ層や業者ではなく、一枚一枚の状態を丁寧に愛着を持って管理したいと感じている「趣味のコレクター層」に寄り添いたいため。

理由2： 既存の英語圏アプリのUIに馴染めず、日本語で「盤質表記」や「国内盤特有の項目」を管理したいユーザーを対象とするため。

## サービスの利用イメージ
ユーザーは、レコードを購入した際や自宅で鑑賞・メンテナンスをする際にスマホからアプリを開きます。 アルバム名や型番を入力すると、外部APIからジャケット画像や基本情報が自動補完されるため、入力のストレスがありません。そこに「帯の有無」や「クリーニング実施日」を数タップで記録。 外出先のレコード屋で「これ持ってたかな？」と迷った時には、所有リストを即座に検索して「ダブり買い」を防止できます。また、友人に自分のコレクションを画面で見せることで、音楽を通じたコミュニケーションをより深いものにします。

## ユーザーの獲得について
SNSでの共有： X (旧Twitter) の「#レコードのある生活」といったコミュニティで、管理の悩みを抱える層へ発信。

専門メディアへのアプローチ：
レコード収集の楽しさを発信している個人ブログやnoteにて、便利な管理ツールとして紹介。

シェア機能による拡散： 自分のコレクションを画像としてシェアできる機能を実装し、音楽好きの間での自然な認知獲得を目指します。

## サービスの差別化ポイント・推しポイント
国内市場に特化した項目： 既存アプリではメモ欄に書くしかなかった「帯の有無」「マトリックス番号」「購入店」「クリーニング履歴」などを独立した項目として管理でき、検索やソートも可能。

徹底した入力負荷の軽減： Discogs APIによる自動補完を活用。手入力を最小限に抑えつつ、コンディション選択などはスマホ片手で完結する直感的なUIを提供。

「自分専用」のクローズドな安心感： SNS的な繋がりによる「見せびらかし」を主目的とせず、あくまで自分の資産を静かに、かつ詳細に整理することに集中できる体験。

## 機能候補
<MVPリリース時>
ユーザー登録・ログイン機能： 自分のコレクションをプライベートに保持。

レコード登録・編集・削除機能（CRUD）：

API検索（Discogs API）および手入力による登録。

独自項目（盤質、ジャケット状態、帯有無、購入日、クリーニング済フラグ）の記録。

コレクション一覧表示・検索・ソート機能： アーティスト別、購入日順、ジャンル別など。

ジャケット画像表示： APIから取得した画像の表示およびActive Storageを利用した画像保持。

<本リリースまで>
欲しい物リスト（Wishlist）： 所有物とは別に、今後手に入れたい盤をリスト化。

統計ダッシュボード： 所有盤のジャンル比率や年代別分布をグラフで可視化（Chart.js等）。

CSVインポート・エクスポート： 既存のExcelやスプレッドシート管理からの移行を支援。

バーコードスキャン機能： スマホカメラでJANコードを読み取り、APIから自動検索する機能。

## 使用する技術スタック
<実装イメージ（CRUD以外）>
外部API連携（Discogs API）： faraday を使用し、APIからレコード情報を取得。ユーザーの入力負荷を軽減します。

画像ストレージ（Cloudinary）： APIから取得したジャケット画像のキャッシュや、ユーザーがアップロードする画像の保存に使用。
